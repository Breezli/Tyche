## 配置参数

| 参数       | 类型              | 默认值      | 说明        |
| ---------- | ----------------- | ----------- | ----------- |
| type       | `'salvage'`       | `'salvage'` | 卡片类型    |
| translateX | `number`/`string` | `0`         | X轴平移     |
| rotateY    | `number`/`string` | `0`         | Y轴旋转角度 |
| ...        | ...               | ...         | ...         |
