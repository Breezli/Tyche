单个导入

Tyche 提供了基于 ES Module 的开箱即用的 Tree Shaking 功能。

vue
<template>
  <er-button>我是 Button</er-button>
</template>
<script>
  import { ErButton } from ' er-element'
  export default {
    components: { ErButton },
  }
</script>
