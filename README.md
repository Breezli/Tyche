### 项目简介

采用 **Monorepo** 架构策略

技术栈：**Vue 3 + TypeScript + Vite**

测试工具：**Vitest + Storybook**

打包工具：**Rollup + Vue-Test-Utils**

项目部署：**VitePress**站点生成 + **GitHubPages**页面托管 + **github actions CI/CD** 自动化部署

### 测试开发

VitePress组件库项目启动

```
pnpm run docs:dev
```

Storybook组件预览测试

```
pnpm run story
```

Vitest单元文件测试

```
pnpm run test
```
